Once, the application is installed, the user needs to create an account to log into the app. The user registration can be done in two ways:
  • Registration with email
  • OTP(One Time Password) Login

During email registration, the user needs to provide his/her email address, a secure password along with first name and last name. This email address will be used to login later.

If the user is an existing user, the user just needs to enter his/her email for login. The following screens will be shown successively:

In case of OTP login, the user has to provide his/her phone number. An OTP is sent to the phone and the app automatically verifies the OTP. Once everything is verified, an account is created and then user can login.

For the user to log out of his/her account, the "Logout" option from the right Navigation Menu is to be clicked.
