The user can customize the application according to his/her preference in the following two ways:
  • Changing the app's default language
  • Changing the app's default mode

As of now, the app supports two languages: English, which is the default language of the app, and Hindi. The "Language" option of the navigational menu lets the user change app's language.

In the same manner, there's an option named "Night mode" in the navigational menu for changing app's mode. The toggle button is just needed to be clicked to change the app's default mode i.e., "Light". However, this "Night" mode has been provided to relieve users' eyes in case they are using the app in a low light or dark environment.
