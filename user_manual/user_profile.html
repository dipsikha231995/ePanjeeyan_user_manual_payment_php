The users of the application can view and update his/her profile once they create an account and log into the app. The "My Profile" screen has been designed for that purpose. Users have the following ways to customize his/her profile:
  • They can update the username.
  • They can update the email address, provided "Email login" is used.
  • They can change their existing password, provided "Email login" is used.
  • If "OTP login" is used, they can update their phone number.

An "edit" icon is present in the top-right corner of the My Profile screen, a user can update changes by clicking it. To update anything, for example, email address, the user needs to write in the new email address and then press "edit" icon right beside the input field. When it is done the "edit" icon changes to a "check" icon and then a pop-up shows up informing that the operation was successful.

In case of updating the existing password, the user has to provide a new password and also a confirm password. Both the passwords need to be strong enough (i.e., a mixture of letters, numbers, etc and atleast 6-character long) and must be the same.
