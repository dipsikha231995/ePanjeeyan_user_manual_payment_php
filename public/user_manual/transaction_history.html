To review all the transactions made by a particular user, the user needs to go to "Transaction History" screen. There, he/she can see a list all the transactions organized in reverse choronological order(i.e. from newest (most relevant) to oldest (least relevant)). At first glance, the user can notice Challan Date, GRN No. and Amount for every transaction.

Upon tapping on a particular transaction card, the user will move to a new "Transaction Details" screen, where he/she can review the details fo that particular transaction. The information presented here includes:

  • Date of the Transaction
  • GRN No.
  • Office Name
  • Total Amount
  • Status of the Transaction
  • And, the list of scheme names and the respective amount for each scheme

If the status of a transaction is "Success", then, the user can download the challan associated with the transaction.

If the status is "Pending", there will be a button-"Verify Paymet with Bank", to verify the current transaction status.

In case of unsuccesssful transactions, the Status will be "Fail". In that case, the user has to repeat the whole payment process again.

To filter the list of transactions, the user has to tap on the the Filter icon in the App bar. On doing so, Filter dialog box shows up, where the user can filter the transactions on the basis of year and month(s).
