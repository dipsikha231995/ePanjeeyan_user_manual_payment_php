After every successful payment, a challan for the same is generated. Users can dowload the challan to their phones right away or if they wish, they can downlod later. "Search Challan" screen has been designed for that purpose. There, a user can search a challan by the GRN Number of that paricular transaction. If a transaction is successful, there will be a download button to download the respective challan.

A "Seach Challans by Voice" feature has also been included to leverage Google's Voice Search. Users have to tap on the mic icon, next to the search field to use voice serach feature. This will open up the voice search widget and users can dictate the GRN no.
