The users of the application can update his/her profile once they create an account and log into the app. The "Edit Profile" screen has been designed for that purpose. Users have the following ways to customize his/her profile:
  • They can update the username.
  • They can update the email address, provided "Email login" is used.
  • They can change their existing password, provided "Email login" is used.
  • If "OTP login" is used, they can update their phone number.

To update anything, for example, email address, the user needs to write in the new email address and then press "done" mark(the check symbol) right beside the input field. If everything checks out then a pop-up shows up informing that the operation was successful.

In case of updating the existing password, the user has to provide a new password and also a confirm password. Both the passwords need to be strong enough(i.e. a mixture of lettes, numbers etc and alteast 6-character long) and must be the same.

Some security-sensitive actions—such as setting a primary email address, and changing a password—require that the user has recently signed in.
